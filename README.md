# 목표 포트폴리오

개인 투자 포트폴리오 관리 및 분석 페이지.

**[→ 바로 사용하기](#)**

## 주요 기능

- 종목·티커·유형·목표 비중 테이블 (정렬·검색)
- 비중 원형 차트 / 섹터별 도넛 차트
- ETF 클릭 시 상세 정보 및 구성 종목 표시
- 장기 시나리오 그래프 (11% / 15% / 17%, 일반·로그 스케일)
- 역사적 위기 스트레스 테스트 (닷컴·금융위기·코로나·금리충격)
- 몬테카를로 시뮬레이션 10,000회 — 연 15% CAGR 달성 확률 계산
- 찰리 멍거 렌즈 — 투자 철학 정리 및 포트폴리오 평가
- 월별 포트폴리오 분석 보고서 (1~3월)
- 월간 기록 트래커 — 평가금액·수익률 기록 및 차트 (localStorage)

## 기술 스택

Vanilla HTML/CSS/JS · Chart.js · GitHub Pages · localStorage

## 데이터

모든 기록 데이터는 브라우저 `localStorage`에만 저장됩니다. 서버 전송 없음.
